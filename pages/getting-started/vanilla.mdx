# Vanilla JS

This guide covers how to integrate Mysterian with a Vanilla JS application. By following these steps, you can leverage powerful AI capabilities directly in your web applications without the need for any frameworks.

## Getting Started

To use Mysterian in a Vanilla JS project, you'll need to include the Mysterian library and set it up to append to the `window` object as `window.mysterian`.

### Including the Mysterian Library

You can include the Mysterian library via a script tag in your HTML file. Here’s an example of how to do it:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Vanilla JS Mysterian Integration</title>
</head>
<body>
  <div id="app">
    <h1>AI-Powered Vanilla JS App</h1>
    <input type="text" id="user-input" placeholder="Ask AI something...">
    <button id="ask-button">Ask AI</button>
    <p id="response"></p>
  </div>

  <!-- Include the Mysterian library -->
  <script src="https://cdn.mysterian.com/mysterian.min.js"></script>
  <script src="app.js"></script>
</body>
</html>
```

### Initializing Mysterian

In your `app.js` file, you can initialize Mysterian and set up the necessary functions to handle AI interactions.

```javascript
// app.js
document.addEventListener("DOMContentLoaded", () => {
  // Initialize Mysterian
  window.mysterian.initializeModel('Llama7B')
    .then(model => {
      console.log('Model loaded successfully:', model);
      setupUI(model);
    })
    .catch(error => {
      console.error('Error loading model:', error);
    });

  function setupUI(model) {
    const inputField = document.getElementById('user-input');
    const askButton = document.getElementById('ask-button');
    const responseParagraph = document.getElementById('response');

    askButton.addEventListener('click', () => {
      const userInput = inputField.value;
      if (userInput) {
        window.mysterian.runInference(model, userInput)
          .then(response => {
            responseParagraph.textContent = `Response: ${response}`;
          })
          .catch(error => {
            responseParagraph.textContent = `Error: ${error.message}`;
          });
      }
    });
  }
});
```

## Handling Errors

Handling errors is crucial for providing a robust user experience. Here’s an example of how to handle errors in your Vanilla JS application:

```javascript
// app.js
document.addEventListener("DOMContentLoaded", () => {
  // Initialize Mysterian
  window.mysterian.initializeModel('Llama7B')
    .then(model => {
      console.log('Model loaded successfully:', model);
      setupUI(model);
    })
    .catch(error => {
      console.error('Error loading model:', error);
      alert(`Error loading model: ${error.message}`);
    });

  function setupUI(model) {
    const inputField = document.getElementById('user-input');
    const askButton = document.getElementById('ask-button');
    const responseParagraph = document.getElementById('response');

    askButton.addEventListener('click', () => {
      const userInput = inputField.value;
      if (userInput) {
        window.mysterian.runInference(model, userInput)
          .then(response => {
            responseParagraph.textContent = `Response: ${response}`;
          })
          .catch(error => {
            responseParagraph.textContent = `Error: ${error.message}`;
          });
      } else {
        alert('Please enter a question before asking the AI.');
      }
    });
  }
});
```
